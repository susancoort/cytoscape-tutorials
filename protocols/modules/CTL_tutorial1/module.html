Cytargetlinker Tutorial 1

Extend a set of microRNAs with regulatory information

Introduction

  • CyTargetLinker provides a quick and extensive enrichment of biological networks in Cytoscape
  • The visualization options allow biological interpretation of complex regulatory networks in a graphical way
  • In this tutorial a step-by-step explanation is given on how to extend a list of microRNAs with predicted and validated microRNA-target interactions (MTIs)

Step 1a: Import a set of microRNAs into Cytoscape

  • First you will need to create a biological network in Cytoscape 3.0 and higher containing your microRNAs of interest.
  • The datafile should contain at least two columns, (i) the microRNA name and (ii) the corresponding miRBase accession number.
  • An example datafile, in which two microRNAs known to be involved in obesity are listed, can be found here
  • Open the datafile containing the microRNAs in Cytoscape via File -> Import -> Network ->File.
  • First you select the file that you want to load into Cytoscape. Thereafter, a panel becomes visible were you should select the microRNA names as Source Nodes and miRBase acc-number as Source Node Attribute, as shown on your right.

Step 1b: microRNA network

The resulting biological network of two microRNAs from the example data, as shown in the figure.

Step 2a:What are linksets?

  • A linkset is a network containing regulatory interactions that are usually derived from an online interaction resources
  • The networks are stored in XGMML (the eXtensible Graph Markup and Modeling Language)format, which is supported by Cytoscape
  • Each interaction consists of two nodes, a source and target biomolecule, connected through one directed edge.
  • Several linkset for different species and interaction types are provided
  • However CyTargetLinker is not limited to the provided linksets. A user can also create their own linksets
  • A detailed description on how to create your own linkset will soon be available as a tutorial

Step 2b: Download the microRNA-target gene linksets

  • Before starting CyTargetLinker you need to download the linksets here and store them in a directory
  • The linksets provided on the CyTargetLinker website are species-specific
  • In this tutorial the human linksets are needed
  • Download the complete Homo Sapiens collection

Step 3a: Built regulatory network in Cytoscape

  • Install the CyTargetLinker app via the app manager in Cytoscape version 3.0 and higher
  • Open the CyTargetLinker in Cytoscape via Apps -> CyTargetLinker -> Extend network
  • Thereafter the CyTargetLinker selection panel is displayed

Step 3b: Built regulatory network in Cytoscape

      Select the user network. This is the biological network you want to extend with regulatory information. Here we select the human microRNAs
      Select the network attribute. This is the column containing the identifier which is supported by CyTargetLinker. The supported identifiers are listed in overview of the linksets here . For the microRNAs we recommend to use the miRBase accession numbers
      Select the directory containing the species specific linksets which you downloaded in step 2b
      Select the direction of the interaction. The user can decide how he wants to extend the network, i.e., a) add targets, b) add sources or 3) both (default). Here we add only the targets of the microRNAs

Step 3c: Built regulatory network in Cytoscape

  • Click ok en thereafter select the linksets containing microRNA-target information
  • In this case we have three resources containing microRNA-target interactions, i.e., microCosm, miRTarBase and TargetScan

Step 3d: Regulatory network in Cytoscape

  • The extended network is now visualized in a new Cytoscape network
  • To visualize the microRNA names and gene names in the network go to View -> Show Graphical Details
  • In the network microRNAs and target genes are defined as grey circles and petrol cycles, respectively
  • To show the CyTargetLinker results panel go to Apps -> CyTargetLinker -> Show results panel
  • The predicted MTIs are visualized in red (microCosm: 1543) and blue (TargetScan: 2550) and the validated MTIs in purple (miRTarBase: 26), as shown in the results panel on the right hand site.
  • The color of the edges can be adapted in the results panel by clicking on the color

Step 4a: Adapt visualization options

The visualization options in CyTartgetLinker enable the investigation of the network in detail

Hide and show functionality

  • It is possible to hide or show specific linksets
  • Here we are interested in only the validated MTIs. Go to the results panel and hide the linksets containing predicted MTI, i.e., TargetScan and MicroCosm, by clicking on the dropdown menu.
  • Now only the linksets from miRTarbase are shown in the network and the layout is immediately adapted

Step 4b: Adapt visualization options

The visualization options in CyTartgetLinker enable the investigation of the network in detail

Threshold functionality

  • It is possible to show interactions that are present in at least a specific number of linksets
  • Here we are interested in MTIs that are present in at least 2 linksets
  • Go to the CyTargetLinker-panel, make sure all the linksets are shown, and set the overlap threshold to 2
  • Now the overlap between the linksets is shown
  • To adapt the layout go to Layout -> Apply preferred layout
  • Per linkset it is indicated first how many interactions are still present when applying a threshold of 2 and the second number represent the total number of interactions were added to the original network
